# Divide and Conquer Min-Max

Implementation of the Divide and Conquer technique to find the minimum and maximum elements in an array, along with a comparison against the naive approach.

## Features

- Finds the minimum and maximum values using Divide and Conquer.
- Compares the number of comparisons with the naive linear approach.
- Includes a performance analysis for different input sizes.

## Requirements

- Python 3.x

## Usage

```bash
python min_max.py
```
