# Dijkstra's Shortest Path Algorithm

Implementation of Dijkstra's Single Source Shortest Path algorithm in Python using a min-heap (priority queue).

## Features

- Computes the shortest distance from a source vertex to all other vertices.
- Reconstructs the shortest path to each vertex.
- Uses a min-heap for efficient shortest path computation.
- Includes validation for invalid source vertices and negative edge weights.

## Requirements

- Python 3.x

## Usage

```bash
python dijkstra.py
```
