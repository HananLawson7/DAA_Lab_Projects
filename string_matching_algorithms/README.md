# String Matching Algorithms

A Python implementation of three classical string matching algorithms with a simple performance comparison.

## Algorithms

- Naive String Matching
- Knuth–Morris–Pratt (KMP)
- Rabin–Karp

## Features

- Finds all occurrences of a pattern in a given text.
- Counts character comparisons for each algorithm.
- Compares algorithm performance on randomly generated input.

## Requirements

- Python 3.x

## Usage

```bash
python string_matching.py
```
